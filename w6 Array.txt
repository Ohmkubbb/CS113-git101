Array
    คือ กลุ่มของตัวแปรที่ถูกจัดร่วมกัน
โดยข้อมูลภายใน Array นั้นจะต้องเป็นประเภทเดียวกันทั้งหมด
เราสามารถประกาศตัวแปร Array ได้ดังนี้
char c[10];
โดยใช้เครื่องหมาย [] เพื่อบ่งบอกว่าเป็น Array และ ข้อมูลข้างในเป็นการบอกขนาดของ Array

การเรียกใช้
    ใน Array นั้นจะมี index เพื่อบอกลำดับของข้อมูล โดยจะเริ่มจาก 0 จนถึง index ที่ size - 1
ex. c[10] เราจะสามารถเรียก index ได้แค่ 0 ถึง 9 เท่านั้น

ตั้งค่าเริ่มต้นของ Array
เราสามารถตั้งค่าเริ่มต้นของ Array ได้
ex. char c[5] = {'W','e','e','k','6'};
หรือ int A[10] = {0,1,2,3}; โดย index ที่เหลือจะมีค่าเป็น 0

เราสามารถใช้ลูบเพื่อเข้าถึงตัวแปรภายใน Array ได้ 
ex.
#include <stdio.h>

int main()
{
   char c[5] = {'W', 'e', 'e', 'k', '6'};
   for (int i = 0; i < 5; i++)
   {
      printf("%c", c[i]);
   }
}

ในการประกาศ Array เราควรที่จะใช้ ค่าคงที่ หรือ define เพื่อให้มีค่าไม่เปลี่ยนแปลงตลอดโปรแกรม
ex. #define arraySize 5 //ค่าของ arraySize จะเป็น 5 ทั้งโปรแกรม
หรือ const int arraySize = 5;

ส่ง Array เข้า Function
-การส่ง Array เข้าไปโดยปกติ จะเป็นการส่งแบบ pass by value (เพื่อจบการทำงานของ Function ค่าไม่เปลี่ยนแปลง)
-การส่ง Array เข้าไปนั้น คอมไพเลอร์นั้นไม่สามารถรับรู้ขนาดได้ด้วยตัวเอง ทำให้เราต้องส่ง ขนาดให้กับ Function ด้วย
ex. int addOne(int array[],int size); //int array[] เป็นการส่ง array,int size เป็นการส่งขนาด

pass by value / pass by reference
-pass by value นั้นเป็นการ copy ตัวแปรไปให้ Function และเป็นตัวแปร local ทำให้ไม่ส่งผลถึงตัวแปรตั้งต้น
-pass by reference เป็นการส่งตำแหน่งของตัวแปร ทำให้เป็นตัวแปรเดียวกับตัวแปรตั้งค่าทั้งในและนอก Function ทำให้หากมีการเปลี่ยนแปลงจะเปลี่ยนแปลงทั้งหมด

Array 2D
-Array 1 มิติ จะเป็นเส้น
-Array 2 มิติ จะเป็น Matix
โดยเราสามารถประกาศได้ดังนี้
ex. int A[2][3]; // ซึ่งก็คือ มี 2 แถว 3 หลัก

การส่ง Array 2 มิติ ให้กับ Function
เราต้องกำหนด จน หลักให้ชัดเจน
ex. int addOne (int array[][3],int row,int col); // ครบ 3 หลัก ขึ้น แถวใหม่
